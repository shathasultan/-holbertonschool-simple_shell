# Simple Shell

This is a simple shell implementation in C that allows users to execute commands. It is a basic version of the UNIX shell, designed to handle simple commands without arguments, as well as basic error handling and file operations.

## Requirements

- GCC (GNU Compiler Collection)
- Linux/Unix environment

## Compilation

To compile the simple shell, use the following command:

```bash
gcc -o simple_shell simple_shell.c
